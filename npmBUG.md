## npmBUG(可能是淘宝镜像源有问题)

## 注意node版本!!!

#### node-sass安装

镜像安装node-sass 

```
npm install node-sass@latest --sass-binary-site=https://npm.taobao.org/mirrors/node-sass

```

#### Node Sass version 7.0.0 is incompatible with 4.0.0 

```
npm uninstall node-sass
npm i -D sass
npm install node-sass
```
